O que é?

Reveal.js é um framework de código aberto para a criação de apresentação de slides no navegador utilizando HTML.

Como funciona?

O framework reveal.js está disponível em https://github.com/hakimel/reveal.js/.

Para utilizá-lo, basta importá-lo no seu HTML e escolher um tema

O reveal.js permite criar os slides apenas com código HTML, incluindo até mesmo temas e estilos

Os slides estão inseridos dentro de um <div> slides, que por sua vez está dentro de um <div> reveal

Cada slide é uma <section> do HTML, e o conteudo do slide também é gerado a partir de código HTML 




	

		<section>
			
		</section>

Para gerar o conteudo, utilizamos texto e tags HTML (como <p>, <a>, <h1>, ...)

O que o reveal.js pode fazer?

Além de criar a apresentação, o reveal.js permite personalizar vários pontos da apresentação de slides para deixá-la ainda mais interativa

Vejamos alguns exemplos:

Slides verticais

Podemos inserir slides verticalmente! 


<section>
	<section>
		
	</section>
	<section>
		
	</section>
</section>

Este tipo de slide é útil para dar maiores detalhes na explicação sobre algum assunto

Porém, se os detalhes não são necessários, podemos pular estes slides detalhados!

Atalhos interativos

O reveal.js também oferece algumas funcionalidades adicionais através de alguns atalhos

Esc

Abre um mapa dos slides da apresentação

Alt + clique

Aumenta/diminui o zoom em um slide

S

Abre o modo de apresentação, que possui algumas informações úteis para quem apresenta

Informações como próximo slide da apresentação, tempo decorrido da apresentação, hora atual, e algumas notas como estas!

B

Esconde os slides, para interromper a apresentação

Temas

O reveal.js disponibiliza algumas opções de temas para a apresentação, logo não precisamos mexer com css!

Black (default) - White - League - Sky - Beige - Simple - Serif - Blood - Night - Moon - Solarized

Background de um slide

Podemos escolher uma cor para o background de um slide em específico 


<section data-background="#ffb3b3">
	
</section>

Ou escolher uma imagem como background 


<section data-background="[url da imagem]">
	
</section>

Ou escolher uma imagem e repeti-la para preencher todo o background 


<section data-background="[url da imagem]"
	data-background-repeat="repeat"
	data-background-size="200px">
	
</section>

Ou escolher um vídeo como background 


<section data-background-video="[url do arquivo .mp4/.webm]">
	
</section>

Ou escolher uma gif como background 


<section data-background="[url da gif]">
	
</section>

Transições

O reveal.js também disponibiliza algumas opções de transições de slide, como:
None - Fade - Slide - Convex - Concave - Zoom

Transição de um slide

Para um mesmo slide, também podemos aplicar tipos diferentes de transição para o conteudo e para o background

Exemplo: slide + zoom 


<section data-transition="slide"
	data-background="#6666ff"
	data-background-transition="zoom">
	
</section>

Exemplo: zoom + fade 


<section data-transition="zoom"
	data-background="#ffffe6"
	data-background-transition="fade">
	
</section>

Exemplo: fade + slide 


<section data-transition="fade"
	data-background="#ff0066"
	data-background-transition="slide">
	
</section>

Configurações adicionais

Podemos utilizar data-state="class" em um slide para que a classe "class" seja adicionada ao slide

Desta maneira, podemos fazer algumas alterações adicionais sobre o estilo dos slides

E ainda mais configurações

Ao inicializar o reveal.js, podemos modificar algumas das configurações padrão do framework para modificar visualmente a apresentação

Exemplos:

  • transition: tipo de transição da apresentação
  • center: centralizar verticalmente o conteudo dos slides
  • controls: mostrar as setas na apresentação
  • progress: mostrar a barra de progresso

Mais exemplos:

  • keyboard: permitir a navegação pelas setas do teclado
  • mouseWheel: permitir a navegação pelo botão de rolagem do mouse
